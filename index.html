Installation Guide

Curseforge installation guide

  1. Download the zip file above
  2. Open the curseforge app
  3. Click Import on the top left of the screen
  4. Then click "Choose .Zip file"
  5. Select the zip file you just downloaded
  6. And thats it well done

Modrinth installation guide

  1. Download the mrPack above
  2. Open the modrinth app
  3. Click Create new instance on the middle left of the screen
  4. now choose From File
  5. and click Import from file
  6. navigate to the file you downloaded earlier
    and double click it

Prism installation guide

  1. Download the zip file above
  2. Open Prism Launcher
  3. Click add instance (top left)
  4. Select Import
  5. Then click Browse
  6. and double click the file you downloaded earlier
About Gearbound

Hello there ^-^
Gearbound is a Create focused community,
aiming to bring automatisation to the world of Minecraft,
We started as a group of friends who slowly over the years all met on this smp,
trying out modpacks one after the other and
then slowly started customizing the packs we played
customizing them how we liked

and now here we are also offering you a place to try out all your ideas
